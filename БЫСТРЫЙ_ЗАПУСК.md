# 🚀 БЫСТРЫЙ ЗАПУСК TENDERAI

## Если интерфейс не загружается:

### 1. Проверить что запущено:
```bash
ps aux | grep -E "vite|node.*3002" | grep -v grep
```

### 2. Убить все процессы:
```bash
pkill -f vite && pkill -f node
```

### 3. Запустить все сразу:
```bash
./start-all.sh
```

### 4. Проверить что работает:
- Интерфейс: http://localhost:5173 (или другой порт)
- WhatsApp API: http://localhost:3002

## Если скрипт не найден:
```bash
chmod +x start-all.sh
```

## Проверка API:
```bash
curl http://localhost:3002/api/whatsapp/status/comp1
```
Должен вернуть: `{"success":true,"status":"ready","connected":true...}`

## Логи:
- `interface.log` - ошибки интерфейса
- `whatsapp-api.log` - ошибки API

## ⚠️ ВАЖНО:
WhatsApp компонент ОБЯЗАТЕЛЬНО требует API на порту 3002!
Без него интерфейс зависает на загрузке.
